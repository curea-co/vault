# 용어 정의집

이 문서는 `learning` 폴더 내 문서에서 사용되는 주요 용어를 정의합니다. 새로운 용어가 등장할 때마다 이 문서를 업데이트하여 일관된 용어 사용을 유지합니다.

---

## 사용 규칙

1. **용어 등록 기준**: 문서에서 처음 등장하는 기술 용어, 약어, 개념
2. **형식**: `**한글명** (English) - 정의`
3. **정렬**: 가나다순 (ㄱ-ㅎ), 이후 알파벳순 (A-Z)
4. **참조 방법**: 문서 내에서 `[[GLOSSARY#용어명]]`으로 링크
