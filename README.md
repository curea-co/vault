# Curea Vault

팀의 학습 지식과 업무 경험을 기록하고 연결하는 Obsidian 기반 지식 그래프입니다.

## 구조

| 영역 | 설명 |
|------|------|
| `learning/` | 기술 개념, 튜토리얼, 트러블슈팅 등 학습 기록 |
| `work/` | 프로젝트별 의사결정, 설계, 회고 등 업무 기록 |

각 영역은 `_meta/`(RULES, INDEX, GLOSSARY)와 `_templates/`로 구성되어 일관된 품질을 유지합니다.

## 시작하기

1. 이 레포를 클론합니다.
2. Obsidian에서 클론한 폴더를 볼트로 엽니다.
3. 문서 작성 시 해당 영역의 `_meta/RULES.md`를 먼저 확인합니다.
